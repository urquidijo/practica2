Página principal del Grupo 14

Esta página fue desarrollada como parte de la práctica de la materia Tecnología Web SA. El objetivo principal es aplicar la sintaxis de HTML5 y utilizar etiquetas modernas para estructurar correctamente una página web.

En este sitio se incluye una página de presentación personal, un curriculum vitae individual, una sección multimedia con videos y audios, una tabla de estadísticas de un municipio y una tabla de compatibilidad entre navegadores y etiquetas HTML5.

Etiquetas HTML5 utilizadas

  • <header> para el encabezado.
  • <nav> para el menú de navegación.
  • <main> para el contenido principal.
  • <section> para dividir el contenido.
  • <article> para contenido independiente.
  • <video> para insertar videos.
  • <audio> para insertar audios.
  • <table> para mostrar datos organizados.
  • <footer> para el pie de página.
