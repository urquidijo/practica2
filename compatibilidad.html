Preguntas de la práctica

¿El código funciona en ambos sistemas operativos y navegadores?

Sí, el código fue desarrollado con HTML5 y CSS3, por lo que funciona correctamente en sistemas operativos Windows y Linux/Fedora. Las páginas fueron diseñadas con etiquetas estándar para permitir una correcta visualización en distintos navegadores.

¿Qué diferencias se encontraron entre los diversos navegadores?

Las principales diferencias se encuentran en la apariencia visual de los controles de audio y video, el tamaño de algunos márgenes por defecto, el estilo de los botones y la forma en que cada navegador renderiza ciertos elementos del formulario o multimedia.

¿Cuál es la solución para estas diferencias?

La solución consiste en usar CSS personalizado, validar el código con W3C, utilizar formatos multimedia compatibles como MP4 para video y MP3 para audio, además de realizar pruebas en varios navegadores antes de subir el sitio al servidor.
